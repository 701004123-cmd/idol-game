# Raylib VS Code Starter Template 🎮

A minimal, boilerplate-free starter template for developing 2D and 3D games using **C++** and [Raylib](https://www.raylib.com/) within **Visual Studio Code**. Designed specifically for student and hobbyist developers to build and run code instantly.

---

## 🛠️ Prerequisites

Before launching the project, ensure your environment meets these basic setups:

* **Visual Studio Code:** Download and install the latest version of [VS Code](https://visualstudio.com).
* **C/C++ Extension:** Install the official [Microsoft C/C++ Extension](https://visualstudio.com) within the VS Code Extension Marketplace (`Ctrl+Shift+X`).
* **Raylib Compiler & Library:** A working compiler suite (such as MinGW for Windows) with [Raylib installed](https://www.raylib.com/) globally or targeted on your `C:\raylib` directory.
